# Awesome LLMs for Time Series Analysis

A curated collection of papers, code, datasets, and resources on large language models (LLMs) for time series forecasting, classification, fault diagnosis, anomaly detection, and prognostics.

## Overview

This repository focuses on how numerical time series are adapted to pretrained language models for various downstream tasks, especially in industrial and real-world scenarios.

### Main Topics
- Time series forecasting
- Time series classification
- Fault diagnosis
- Anomaly detection
- Remaining useful life prediction
- Prompting, patching, adapter, reprogramming, and multimodal alignment

## Taxonomy

### By task
- Forecasting
- Classification
- Fault Diagnosis
- Anomaly Detection
- RUL / Prognostics

### By adaptation strategy
- Numerical-to-text prompting
- Patch embedding
- Adapter-based integration
- Reprogramming / cross-attention alignment
- Multimodal LLMs
