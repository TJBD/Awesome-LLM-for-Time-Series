## TF-LLM (2026)

**Task:** Forecasting / Classification / Anomaly Detection  
**Core idea:** Joint time-domain and frequency-domain feature modeling before LLM adaptation.  
**LLM adaptation:** Prompt + feature embedding  
**Backbone:** LLM  
**Strength:** Strong feature enhancement before semantic alignment.  
**Limitation:** Framework is relatively complex.  
**My takeaway:** The key innovation may lie more in feature organization than in the LLM itself.
